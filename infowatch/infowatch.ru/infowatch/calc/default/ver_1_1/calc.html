Калькулятор предназначен для расчета стоимости поставки программного продукта InfoWatch EndPoint Security. Калькулятор рассчитывает стоимость с учетом текущей ценовой политики и действующих скидок компании InfoWatch. Данный калькулятор предназначен для использования ТОЛЬКО ВНУТРИ партнерской сети InfoWatch и не подлежит передаче клиентам.

 

Модуль Access Control

Создан для эффективного управления рабочими процессами и грамотного разграничения прав доступа сотрудников к внешним устройствам

 

Модуль Audit

Создан для полного аудита действий пользователей при работе с информацией, в частности, через внешние носители. С помощью этого модуля, можно проверить, кто из сотрудников и когда выполнял те или иные операции с файлами на устройствах (чтение, копирование, создание или удаление) и получить доступ к теневым копиям файлов.

 

Модуль Power Management

Создан для управления мощностью компьютеров. Позволяет по расписанию включать/выключать/переводить в режим ожидания рабочие станции, а также задавать исключения, препятствующие выключению компьютера, – как глобально для всей компании, так и для определенных компьютеров.

 

Модуль Device Encryption

Прозрачное и удобное шифрование данных на внешних носителях без дополнительных усилий и действий. На базе криптоалгоритмов Windows.

 

Модуль Folder Encryption

Прозрачное и удобное шифрование каталогов операционной системы для сотрудников. На базе криптоалгоритмов Windows

 

Модуль Application Control

Создан для контроля доступа сотрудников к приложениям с помощью белых и черных списков. Это гарантирует, что никакая запрещенная программа (например, вирусы, троянские программы, игры и т.д.) не будет установлена или запущена на компьютере предприятия.
